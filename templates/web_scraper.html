How to use the Web Scraper extension

1. Installation and opening

The extension is currently available only for the Chrome web browser. The extension is installed through this page.

After installation, press F12, use the shortcut Ctrl+Shift+i, or go to Settings-More Tools-Developer Tools to open the window where the extension should be located.

2. Setting up the extension to collect data from a source

First, you need to click the Create new sitemap-Create Sitemap button. After that, find an online source. In this example, Amazon reviews for a keyboard are used.

If the source contains multiple pages, you need to find a link that contains the page number, and in the Start URL 1 field, paste the link and replace the number with [first-last]. In this example, the number 2

is replaced with [1-3], as we are collecting reviews from the 1st to the 3rd page.

After creating the sitemap, click the add new selector button. In the Id field, type content (IMPORTANT) and click the multiple checkbox.

Now click the Select button in the Selectors section above the checkbox. The page will now darken, and you'll have the option to select an element. Click the desired element, which in this example is a review,

that element will now be highlighted in red, and then click on another such element. If everything was done successfully, all the desired elements on the page should be highlighted as shown in the image.

Now click the green Done selecting button, and at the bottom of the extension, click the blue Save selector button.

3. Collecting data

In the sitemap section [example], click the Scrape button as shown in the image. A tab will open with options for Request interval and Page load delay, which can be changed as desired. Click the blue Start scraping button.

Once the data collection is complete, the collected columns will be displayed (the extension will offer a Refresh option if the data is not shown).

4. Exporting data

In the sitemap section [example], click the Export data button and click the blue .csv button.

The downloaded file can now be used for analysis.
